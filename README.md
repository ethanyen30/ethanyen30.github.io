# Deployment

Work in the main branch. 
- Run `npm run dev` to start the development server.
- Run `npm run build` to build the production version.
- Run `npm run preview` to preview the production version.

**Normal workflow**: Push to main → GitHub Actions handles everything

**Emergency local deploy** (avoid if possible):
```bash
npm run deploy
